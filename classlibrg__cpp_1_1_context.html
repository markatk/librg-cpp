void librg_cpp::Context::reset ( )

Reset the context settings to default.

Default settings:

  • TickDelay: 32
  • WorldSize: 5000
  • MinBranchSize: 50
  • MaxConnections: 16
  • MaxEntities: 8192

Ideally calling this on both sides will return the same time.
